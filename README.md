**dimension-specific tuning:**
```yaml
nether:
  entity_culling: aggressive (ceiling blocks view)
  render_distance: reduced (can't see far anyway)
  particle_density: high (lava effects important)
  
end:
  entity_culling: relaxed (see endermen from far)
  render_distance: increased (open void)
  particle_density: reduced (less effects)
```

**automatic switching:**
- dimension change detected
- profile switches instantly
- no manual configuration needed
